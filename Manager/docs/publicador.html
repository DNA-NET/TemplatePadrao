Conteúdo 

A maior parte dos serviços de publicação de conteúdo possui interfaces que permitem uma publicação diferenciada de cada conteúdo cadastrado, vários parâmetros podem ser especificados pelo publicador que tem acesso aos conteúdos do site, listados por ordem de inserção.  

Período de Publicação: Este campo define a data e a hora em que o conteúdo será publicado no portal e retirado do mesmo, alterando seu status para arquivado;
Data de expiração: Define a data e hora em que o conteúdo deixará aparecer no site. 

 INCLUIR, BUSCAR E EXCLUIR 

 É possível incluir, buscar e excluir os conteúdos a qualquer momento., bem como mudar o status do conteúdo (publicado, novou ou rascunho), o usuário e as ações.
 Seções associadas, máscara e ações também são informadas na mesma tela.


Registros de conteúdos

Cadastro de Conteúdo

Para cadastrar um conteúdo, basta selecionar a MÁSCARA que deseja utilizar e preencher os dados que pede.

Inserir conteúdo (máscara)

Após selecionar a máscara, o publicador deverá informar os dados abaixo.
Status do conteúdo poderá ser alterado a qualquer momento. (novo, publicado ou rascunho).

Sessões associadas (cada conteúdo tem ORIGEM), sendo assim possível selecionar vários destinos para o mesmo conteúdo.


Publicação para seções associadas:


Em certas ocasiões é interessante poder publicar um mesmo conteúdo em vários locais no site.

Para isso é disponibilizado o campo “destino”, que permite ao editor de conteúdo definir outras seções para o mesmo conteúdo, nas quais ele deseja que um determinado conteúdo também seja publicado. Sendo assim o mesmo conteúdo passa a ser compartilhado em várias seções do site. Origem e destino informam em quantas seções o mesmo conteúdo foi publicado.


Por exemplo: Uma notícia publicada na seção de notícias de uma empresa também pode ser apresentada no canal de Avisos, sendo necessário apenas que o usuário clique sobre o botão da seção e escolher o DESTINO referente a Avisos que contenha a seção de Notícias.

Seções, Navegação, Menus

Organização lógica e hierárquica para classificação dos conteúdos.
Seções representam a arquitetura de informação que define a estrutura completa de navegação do Portal.

Incluir seção

O publicador pode criar novas seções ao site com workflow de perfis. WORK, PUB e RESTRITA

Seção permanente, seção oculta.

Publicação para usuários, grupos ou segmentações: 


Esta funcionalidade permite ao editor publicar cada um dos conteúdos cadastrados para usuários, grupos ou segmentações diferentes. Ainda assim, os usuários referenciados direta ou indiretamente (através de grupos e/ou segmentações) somente visualizarão tais conteúdos se tiverem permissão de acesso na instância de serviço de tal conteúdo. Porém, se o usuário estiver no controle de acesso da instância de serviço direta ou indiretamente mas não estiver na lista de publicação do conteúdo, direta ou indiretamente, ele não poderá o visualizar. Por padrão, o grupo "Todos usuários cadastrados" receberá a publicação dos conteúdos (salvo os usuários que estão neste grupo também estiverem com permissão no controle de acesso).
